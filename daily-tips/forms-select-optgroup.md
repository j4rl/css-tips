# Select and optgroup
Style `select` and group options with `optgroup`.

```css
select {
  appearance: none;
  padding: 0.5rem 2rem 0.5rem 0.75rem;
  border: 1px solid #cbd5e1;
  border-radius: 0.5rem;
}

optgroup {
  font-weight: 600;
}
```

Why it helps: organized dropdowns with consistent styling.
