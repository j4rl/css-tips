# Glossary
Common CSS terms.

- Cascade: The rules by which competing styles are resolved
- Specificity: A weight score for selectors
- Inheritance: How properties flow from parent to child
- Formatting context: Rules for layout in a container
- Stacking context: Isolation for z-index and paint order
- Containing block: The box used to resolve size and position
- Replaced element: Element with external content, like `img`
- Intrinsic size: Size based on content
