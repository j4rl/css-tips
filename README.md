# CSS-tips
CSS tips and a structured guide from fundamentals to advanced techniques.

## Start here
- New to CSS: see `learning-path.md`
- Quick wins and patterns: see `tips.md`
- Core reference: start with `basics.md` and `layout.md`

## Repository map
- `basics.md` - selectors, cascade, box model, units, and sizing
- `layout.md` - normal flow, flexbox, grid, positioning, and containment
- `typography.md` - fonts, text flow, readability, and variable fonts
- `colors.md` - color spaces, contrast, gradients, and tokens
- `responsive.md` - media queries, container queries, and fluid sizing
- `effects.md` - shadows, filters, blend modes, and masking
- `animations.md` - transitions, keyframes, timing, and performance
- `advanced.md` - layers, scope, Houdini, and new features
- `architecture.md` - naming, layering, and scaling strategies
- `performance.md` - rendering cost, paints, and optimization
- `accessibility.md` - focus, motion, and color accessibility
- `resources.md` - references and tools
- `glossary.md` - core CSS terms

## Goals
- Be practical: show the simplest working pattern first
- Be modern: cover cutting edge CSS features and where they fit
- Be reliable: prefer patterns that work across browsers, with fallbacks
